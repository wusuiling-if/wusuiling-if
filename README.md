## Focus

- AI Agent 产品设计：意图理解、工具调用、Agent Skill 封装、上下文治理、结果后处理与失败 case 分析。
- LLM 应用工程：多模型接入、OpenAI 兼容接口、SSE 流式响应、API Key 管理、速率限制、用量日志和调试面板。
- 快速原型开发：使用 AI coding 快速搭建 React / TypeScript / Vite 前端、Node.js / Express 代理、FastAPI 服务和可交互 Demo。
- 记忆与 RAG：长期记忆、向量检索、分层记忆、去重合并、摘要写入和知识库查询接口。
- 多模态算法基础：YOLO、MediaPipe、ResNet / CNN、SLAM、三维视线估计和可微几何投影。
